Changelog Block for Plugins and Themes or anything.

== Description ==

This is a lightweight and simple Gutenberg block for adding changelogs in your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/changelog-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Does this require any coding? =

No, it doesn't.
